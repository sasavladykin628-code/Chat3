# 💬 GitHub Messenger

Простой веб-чат для GitHub Pages.

## Запуск

1. Скачайте `index.html`, `style.css`, `script.js` в одну папку
2. Откройте `index.html` в браузере

## Публикация на GitHub Pages

1. Создайте публичный репозиторий
2. Загрузите все 4 файла
3. **Settings → Pages** → Branch: `main` → Save
4. Откройте `https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ/`

## Возможности

- 💾 Сохранение истории в браузере
- 🎨 Адаптивный дизайн
- ⌨️ Отправка по Enter
- 🗑 Очистка чата

## Важно

⚠️ Сообщения видны **только вам** — это локальный чат
